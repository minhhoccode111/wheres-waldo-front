## **Getting Started**

```bash
HTTPS - git clone https://github.com/minhhoccode111/wheres-waldo-front.git

SSH - git clone git@github.com:minhhoccode111/wheres-waldo-front.git

cd wheres-waldo-front

npm install

npm start
```
